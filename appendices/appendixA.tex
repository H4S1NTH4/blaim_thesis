This appendix provides the operational event fields and simplified scoring logic used by the behavior learning analytics module.

\section*{A.2 Simplified Process Score Formulation}

\begin{equation}
\mathrm{Overall} = \frac{\mathrm{APS} + \mathrm{LD} + \mathrm{AS} + \mathrm{ENG}}{4}
\end{equation}

where:
\begin{itemize}
	\item $\mathrm{APS}$: Active Problem Solving score,
	\item $\mathrm{LD}$: Learning Depth score,
	\item $\mathrm{AS}$: Authenticity score,
	\item $\mathrm{ENG}$: Engagement score.
\end{itemize}

These component scores are derived from transparent heuristics involving pauses, rewrite activity, deletion behavior, paste operations, and session duration.

\section*{A.3 Interpretation Guidance}

\begin{itemize}
	\item \textbf{High authenticity with high struggle}: typically indicates genuine effort and learning in progress.
	\item \textbf{Low authenticity with low edit activity}: requires manual review for possible external assistance.
	\item \textbf{High cognitive load segments}: suitable for targeted pedagogical intervention.
\end{itemize}
