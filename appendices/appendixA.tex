\section{Raw Data Evidence Pack}
\label{app:raw-data-pack}

The detailed testing data used for Chapter 4 is maintained in:

\begin{quote}
\texttt{blaim\_thesis/BLAIM\_TESTING\_DATA\_AND\_METRICS.md}
\end{quote}

This evidence pack contains enrollment records, biometric template summaries, enrollment progress rows, test records, per-student summaries, authentication reruns, archives, timeline events, saved metric snapshots, compact API responses, and submitted code summaries. Raw keystroke arrays are intentionally excluded from the thesis body and retained in database JSON fields to avoid exposing unnecessary biometric data in prose.
